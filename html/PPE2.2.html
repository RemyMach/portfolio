Création du site web Biotourist sur l'application téléphone

  1. Accueil
  2. PPE 2.2

Contexte

Création de l'application téléphone du site web 2.1 sur tous types de téléphones. Cette application devait s'appuyer sur l'API déja fait pour le PPE2.1 et communiquer avec la même base de donnée que celle-ci afin que les utilisateurs puissent utiliser leur compte sur l'application téléphone et sur le site web.

En groupe, nous devions décider de la technologies que nous allions utiliser. Notrre choix c'est vite tourner vers le react-native qui semblait être une technologies intéressante, demandé dans le milieu professionnel et qui nous permettait de faire du cross-platform.

Réalisation

Le plus gros du travail de back-end était déjà fait graçe à l'API réalisé précédemment dans le projet 2.1. Le plus gros problème était le temps qu'il nous restait afin de réaliser l'application sachant que nous n'avions jamais utilsé la librairie react-native. Nous avons donc fait le choix de concevoir les fonctionnalités essentiel à l'application. Nous nous sommes réparties les fonctionnalités, pour ma part j'ai eu:

  1. l'initialisation du projet
  2. Gestion de la navigation entre les différentes vues
  3. Gestion des utilisateurs
  4. Integration de la gestion profil et des messages

J'ai du commencer par me former à cette technologie et j'ai choisi de le faire via le cours openclassroom Développez une application mobile ReactNative Par la suite j'ai donc télécharger le projet initialiser le dépot sur Git et fait mes premier push pour start l'application

Pour la navigation j'en suis très vite venue à l'utilisation de la librairie react-navigation qui me permettait de gérer toute la navigation dans un fichier en ajoutant même quelques animations

Les utilisateurs ont été géré via la librairie asyncstorage qui permet d'enregistrè les utilisateurs renvoyé après la connexion dans la session du téléphone. Celui-ci garde donc en mémoire l'utilisateur jusqu'à ça connexion ainsi que son statut dans lequel celui-ci c'est connecté.

L'intégration du profil et de la gestion des messages c'est fait directement dans l'onglet profil. L'utilisateur peut voir toutes ses conversations et envoyer des messages. Il peut aussi modifié ses informations et son mot de passe.

Outils utilisés

J’ai utilisé pour ce projet PhpStorm pour l’IDE.

Librairie utilisé : Laravel

Langage utilisés : CSS, Javascript

Base de donnée : Mysql

Versionning : Github

Environnement : hébergement sur téléphone.
