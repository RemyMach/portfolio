Mise en place d’un Antivirus au sein du parc informatique

  1. Accueil
  2. Portfolio Pro
  3. antivirus

Contexte

Aucun antivirus n’était alors présent dans le parc informatique, le RSSI à alors décidé d’installer un antivirus global au parc informatique qui avait de très bons avis au sein du milieu professionnel.

Réalisation

Nous avons tout d’abord du installer un SQL Server sur notre machine afin de pouvoir administrer l’ensemble des clés antivirus mais aussi de faire du déploiement à distance de l’antivirus une fois que l’agent serait installé. Une problématique d’environnement c’est très rapidement posé car nous avions des machines avec des systèmes d’exploitation différents(Linux, Windows 10 et Mac OS). Des tests d’installation ont donc suivi, ceux-ci se sont avéré concluant, on suivi des tests d’utilisation pour voir si l’antivirus n’handicapé pas les Ordinateurs client. Nous avons rencontré ici un problème de taille, les filtres interdisant certains type de sites bloquaient des URL nécessaire à la veille technologique de certains collaborateurs mais aussi leur travail. La solution choisit a donc était d’installer l’antivirus sur une machine dans chacun des services pour pouvoir débloquer chaque site par son URL au fur et à mesure sans handicaper l’ensemble du parc informatique mais seulement quelques beta testeurs volontaires. Les beta testeurs nous faisaient parvenir les site sous forme de ticket sur la plateforme zendesk.

Pour le déploiement il m’a fallu écrire des scripts pour Ubuntu et Mac OS afin de pouvoir installé les Agents sans avoir besoin de faire de manipulation, une seule commande suffisait, Un README a été écrit pour cela. Il a fallu ici aussi testé les scripts sur des machine Ubuntu de versions différentes pour vérifier la compatibilité. La procédure a été identique pour les MAC.

Nous avions décidé qu’ont pourrait décrété la fin des tests avec ces béta testeurs quand on ne débloquerais plus d’URL pendant deux semaines. Nous avons par la suite rajouté l’antivirus sur de plus en plus de machines jusqu’à l’entièreté du parc informatique.

Outils utilisés

J’ai utilisé pour ce projet VIM pour écrire les scripts, Git pour le versioning et plus particulièrement le Gitlab interne à l’entreprise.

Langage utilisés : Bash

Base de donnée : SqlServeur
