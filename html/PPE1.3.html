Création d'un site web pour une pizzeria, sandwicherie

  1. Accueil
  2. PPE 1.3

Contexte

En autonomie, nous devions créer un site web avec le langage PHP dans un style de programmation procédurale. La partie back-end devait s’appuyer sur une base de données mysql. Un système de paiement de test devait être intégré et nous devions aussi réaliser un front séduisant à l’aide de Javascript, HTML CSS. Une limite de temps de 6 semaines nous avait été laissé.

Réalisation

Dans un premier temps j’ai fait le choix d’utiliser l’environnement de développement AMPPS qui contient un serveur web, un serveur de base de donnée(Mysql), l’interpréteur de script PHP et aussi l’outil de gestion de base de donnée phpMyAdmin .

La première étape a été la conception de la base de donnée via phpMyAdmin en rentrant toutes les données des produits via l’interface puis la liaison avec la base de donnée et le site.

Une fois la liaison établit tout l’affichage du site a été réalisé grossièrement. Les fonctionnalités d’authentification, de gestion de panier et de paiement via paypal ont ensuite été intégré au site.

Il me restait alors une semaine, j’ai donc décidé de réalisé l’affichage de panier avec AJAX pour la gestion de quantité, et j’ai réutilisé l’AJAX pour la séléction des produits avec leur prix et leur quantité. Il ne m’a alors pas resté beaucoup de temps pour améliorer le front du site. Pour Conclure toute les contraintes ont été remplis et le site était fonctionnel.

Outils utilisés

J’ai utilisé pour ce projet PhpStorm pour l’IDE.

Langage utilisés : PHP, HTML, CSS, Javascript

Base de donnée : Mysql.

Environnement : local sur un environnment AMPPS.
