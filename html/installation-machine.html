Écriture de script pour l'automatisation d'installation de machine

  1. Accueil
  2. Portfolio Pro
  3. Installation machine

Contexte

Il y avait un grand nombre de tâches à effectué à l’arrivé d’un nouveau collaborateur et cela pouvait prendre longtemps. Il a donc été décidé d’écrire des scripts pour améliorer la rapidité de préparation d’une machine mais aussi centralisé l’installation de celle-ci en un script. Durant ce projet je me suis occupé des machines sous Ubuntu et Mac OS.

Réalisation

Il a fallu dans un premier temps mettre en place mes environnement de tests qui étaient des machines clientes MAC (MacBook Pro, MacBook Air) et Ubuntu(Xps 13, Xps 15). Celle-ci devait être configuré comme de nouvelles machines afin de pouvoir effectuer les tests nécessaire comme-ci nous allions par la suite donner ces machines à de nouveaux collaborateurs.

Des scripts ont donc été écrit individuellement pour chaque outils si ces script n’existait pas déjà(Ocs, Outils de gestion RSSI etc…). Ces scripts ont été testé, sur différentes machines avec différentes version d’OS, quand les scripts rendait l’outil parfaitement fonctionnel sur le poste test mais aussi sur tous les autre postes test du système d’exploitation, il était alors décidé que ce script était validé.

Quand tous les scriptes furent validé, la décision de rédigé le script appelant tous les autres fut prise, on lui ajouta quelques fonctionnalité comme changer le nom de la machine de l’utilisateur car son nom était connu à l’avance, se connecter au réseau wifi de Wynd sur la machine. On suivit les mêmes tests sur ce script que sur les autres, puis après sa validation son intégration au processus de préparation de machine officiel. S’en est suivit une évolution de ces scripts avec l’évolution des outils suivant les package d’installation mais aussi les version des systèmes d’exploitation.

Outils utilisés

J’ai utilisé pour ce projet VIM pour écrire les scripts, Git pour le versioning et plus particulièrement le Gitlab interne à l’entreprise.

Langage utilisés : Bash

Machines de test : Macbook Pro 8Go et 16Go, Macbook air 8Go, XPS 13 16Go, XPS 15 16Go.
