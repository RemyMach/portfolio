A quoi sert Bootstrap ?

Bootstrap est souvent comparée à une imensse boite à outils dont la qualité première et de quadrillé une page html pour mieux l'organiser.

Bootstrap apporte tout un tas de fonctionnalités pour développer les sites Web, avec une approche très portée sur le « Responsive Web Design ».

Ainsi les fonctionnalités principales que nous propose Bootstrap sont les suivantes :

  • Un système de grille de mise en page afin de pouvoir agencer les différentes pages de notre site internet. Ce système permet de mettre en place un « Responsive Web Design » très facilement, cela signifie que l’affichage va s’adapter aux différents écrans disponibles aujourd’hui.
  • Des styles CSS pour mettre en forme le texte, les tableaux et les formulaires par le biais de classes.
  • Des styles CSS permettant d’afficher des boutons attrayants et des images design toujours par le biais de classes.
  • Une police de caractères proposant nativement des icônes : http://glyphicons.com.
  • Des composants entier d’interface pour apporter de la dynamique à vos sites Web : des barre de menus, un groupe de boutons, des vignettes d’image…
  • Des plugins JavaScript pour réellement dynamiser l’ensemble : modales, onglets, bulle d’aide…
