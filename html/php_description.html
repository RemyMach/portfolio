Mais qu'est-ce que PHP ?

Le PHP est un langage informatique utilisé sur l'internet. Le terme PHP est un acronyme récursif de "PHP: Hypertext Preprocessor"

Ce langage est principalement utilisé pour produire un site web dynamique. Il est courant que ce langage soit associé à une base de données, tel que MySQL. Il permet entre autres de rendre automatiques des tâches répétitives, notamment grâce à la communication avec une base de données (utilisation la plus courante de PHP). Exécuté du côté serveur (l'endroit où est hébergé le site) il n'y a pas besoin aux visiteurs d'avoir des logiciels ou plugins particulier, mais c'est aussi un langage qui peut s'intégrer dans le code HTML. Il est aussi nécessaire de posséder un serveur Web tel que Apache pour faire fonctionner ces scripts PHP car sans cesse envoyer des fichier via FTP à son hébergeur internet peut s'avérer très lourd. Enfin pour écrire du code nimporte quel éditeur de texte peut-être utilisé.
